一、基本语法

1.标题

#+空格+文本， # 一级标题
几个#几级标题

2.加粗

两个**+文本+，**文本**
例如：Use VS code to write Markdown is a good idea**

3.引用

  1. 英文小箭头>+空格+文本（”>“前面任何字符），>数据结构
  2. 几个>几级

    刘铁猛C#入门学习

    类型定义，语句介绍，选择、循环

刘铁猛C#进阶学习

类的基本认识

4.无序列表

  1. 用-，+，*都可以(符号前面都无文字)
  • 强调无序
  • 加号也可以
  • *号也可以

5.有序列表

  1. 列表1
  2. 列表2
  3. 数字+.+空格，1.列表1

6.TodoList

  1. 介绍：选择框
  2. 样式：
    打勾：- [x] 文本，-[x] 文本
    空：- [ ] 文本，-[] 文本
  • 数据结构
  • 数据库

7.表格

`| 左对齐 | 居中对齐 | 右对齐|

| :—– | :—–: | —-: |
| 冒号在左 | 两边都有 | 冒号在右|`

左对齐 居中对齐 右对齐
冒号在左 两边都有 冒号在右

8.换行

  • 两个以上的空格+回车

    黄行
    黄花

  • 分割线：三个*，***

9.字体

字体 代码
斜体 * *
==高亮== == ==
粗体 ** **
粗斜体 *** ***
删除 ~~ ~~
下划线 <u> </u>
  • 脚注
    谢谢大家观看本篇文章[^1]!!
    [^1]: 谢谢大家?!

  • 代码前后加三个 `

  • 也可以直接在代码前后加 `
